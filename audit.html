Observations

  • Font and input sizes are too small! A modular scale and better vertical rhythm would increase visual flow.
  • Spacing and padding increases will help emphasize/isolate each step in the form fillout process.
  • Bold headers and lighter text will clue the user on how to use the form.
  • Overall look and feel is boring and looks 'technical'. A stronger font, and some small but impactful UI improvements for the field inputs would go a long way!
